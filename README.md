# Vagrant and Ansible

## Atividade MIT DevOps e Cloud SENAI FATESG

* Requirimentos
    - Virtualbox 6.x

Provisionamento de uma instancia AlmaLinux8 usando Ansible playbook para configurar e implementar ambiente com WordPress.

```shell
vagrant up --provider virtualbox
vagrant status
```
